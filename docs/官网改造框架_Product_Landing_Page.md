## 🎨 视觉风格指南

### 配色方案
| 颜色 | 用途 | HEX |
|------|------|-----|
| **主色（蓝紫渐变）** | 主按钮、Hero区 | `#667eea → #764ba2` |
| **辅助蓝** | 链接、次级按钮 | `#3B82F6` |
| **成功绿** | 完成状态 | `#10B981` |
| **警告橙** | 提醒标签 | `#F59E0B` |
| **深色背景** | 侧边栏、代码块 | `#1E293B` |
| **浅色背景** | 卡片、内容区 | `#F8FAFC` |

### 字体
- **标题**: Inter / 思源黑体
- **正文**: -apple-system, BlinkMacSystemFont
- **代码**: Monaco, Menlo, monospace

### 组件风格
- **圆角**: 12px (卡片) / 8px (按钮)
- **阴影**: `0 4px 12px rgba(0,0,0,0.1)`
- **动画**: 300ms ease 过渡
